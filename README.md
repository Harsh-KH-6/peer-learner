# Peer Learner

A modern Next.js + TypeScript starter for a peer-learning web application. This repository contains a frontend built with Next.js (App Router), TypeScript, PostCSS/Tailwind-ready configuration, and a component-driven structure.


- app/ — main Next.js application (App Router) (https://github.com/Harsh-KH-6/peer-learner/tree/main/app)
- components/ — UI components (https://github.com/Harsh-KH-6/peer-learner/tree/main/components)
- hooks/ — React hooks (https://github.com/Harsh-KH-6/peer-learner/tree/main/hooks)
- lib/ — utilities and helpers (https://github.com/Harsh-KH-6/peer-learner/tree/main/lib)
- public/ — static assets (https://github.com/Harsh-KH-6/peer-learner/tree/main/public)
- styles/ — global styles (https://github.com/Harsh-KH-6/peer-learner/tree/main/styles)
- next.config.mjs — Next.js configuration (https://github.com/Harsh-KH-6/peer-learner/blob/main/next.config.mjs)
- package.json — project scripts & dependencies (https://github.com/Harsh-KH-6/peer-learner/blob/main/package.json)
- postcss.config.mjs — PostCSS/Tailwind config entry point (https://github.com/Harsh-KH-6/peer-learner/blob/main/postcss.config.mjs)
- tsconfig.json — TypeScript configuration (https://github.com/Harsh-KH-6/peer-learner/blob/main/tsconfig.json)


---

## Quick Overview

Peer Learner is intended to be a frontend for a peer-to-peer learning platform — connecting learners, organizing study sessions, and sharing resources. The current repository contains the UI scaffolding and component structure to build on.

## Tech stack

- Next.js (App Router)
- TypeScript
- PostCSS / Tailwind-ready (postcss.config.mjs present)
- pnpm for package management (pnpm-lock.yaml present)

## Prerequisites

- Node.js 18 or newer (recommended)
- pnpm (or you can use npm/yarn, but pnpm is used by the lockfile)
